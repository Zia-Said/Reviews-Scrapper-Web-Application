## Reviews-Scrapper-Web-Application

 ### Link (Deployed in Heroku): https://reviewscrapperzia.herokuapp.com/
 
 ### Data Source: https://www.flipkart.com/
 
The main objective of this project use case was to use beautiful soup library to scrap https://www.flipkart.com/ website for reviews on any available item which a user can choose. A whole web application was developed and deployed in Heroku Cloud platform. The entire project files are uploaded in the current github repository with detailed explanation for the code.
